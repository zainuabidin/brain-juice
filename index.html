• Brain Juice is a Java Script Quiz game.
  • Please familiarize yourself with game rules and press start
  • You will be presented with 5 Question
  • Timer has been included so you have 60 seconds in which to answer all questions
  • Wrong answer will result will deduct 10 seconds from the timer
  • WHEN all questions are answered or the timer reaches 0 THEN the game is over
  • You will have an opportunity to save your name after the quiz finished
